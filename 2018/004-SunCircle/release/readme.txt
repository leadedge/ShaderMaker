Installation

copy the .dll file into the [resolume]/plugins/vfx folder of your resolume installation, 
or in the apropriate folder for any ffgl enabled software.

Parameters Description
te radius and width play together, it can be used to achieve various effects,
the width of each segment shall outreach visible area for any circle center

Radius - radius of the circle
Width1 - width of segment 1 
Width2 - width of segment 2
PosX - x position of circle center
PosY - y position of circle center
Divisions - how many divisions applied to the circle
Color 1 - the segment 1 color
Color 2 - the segment 2 color
