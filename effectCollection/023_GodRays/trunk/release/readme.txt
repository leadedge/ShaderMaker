Bias - Input Brightnes for letting light through, only available if "useAlpha" is unchecked
posX/posY - light source position on screen 0,0 is bottom left, 1,1 is topright
density - density of light scattering particles
weight - incoming light weight
decay - decay factor (needs to be high, >0.8 )
exposure - output value 0= no output, 1 = full output
light radius - radius of the light
