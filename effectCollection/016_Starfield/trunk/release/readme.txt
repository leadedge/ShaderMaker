Strobo Sine


Displays up to 100 moving sine dots with a scalable middle point. Full HSV Color range is used for
sine dots, utilising hue shift/scale/rotate desired color setup can be created. The plugin uses an
internal counter, play with the speeds.


Installation
Copy into plugin folder of software.
