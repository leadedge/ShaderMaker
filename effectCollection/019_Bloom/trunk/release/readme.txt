FFGL Bloom Plugin.

Description:
Applies a Brightness Filter to the image, filtering out certain colors, blurs the filtered image and adds it to the input texture.

Parameters:
Blur Amount   - number of rounds when blurring
Brightness    - Brightness Pass Filter 
BrighntessAdd - Amount of brightness added to each pixel
Power         - Brightnes Power Multiplier 
PowerAdd      - Brightnes Power Add
Color Red     - Filter out other colors than just the brightness, allows negative values
Color Green   - "
Color Blue    - "
